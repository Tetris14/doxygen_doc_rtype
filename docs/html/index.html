Project Purpose

The purpose of the project is to recreate the video game called R-Type https://en.wikipedia.org/wiki/R-Type (a famous Shoot 'em game in the 90's) but with a additional multiplayer mode.

Requirements

  • Linux or Windows
  • CMake
  • Compiler C++
  • Asio

How to use

Git clone the project

git clone git@github.com:EpitechPromo2025/B-CPP-500-PAR-5-2-rtype-tristan.gory.git

Create build folder and use cmake For mac users :

mkdir build && cd build
conan remote add bincrafters https://bincrafters.jfrog.io/artifactory/api/conan/conan
conan install .. --build=missing -s build_type=Release -s compiler.libcxx=libc++
cd ..
cmake -B build -S .
cmake --build build
mkdir build && cd build
conan remote add bincrafters https://bincrafters.jfrog.io/artifactory/api/conan/conan
conan install .. --build=missing -s build_type=Release -s compiler.libcxx=libstdc++11
cd ..
cmake -B build -S .
cmake --build build

Binary/Executable are in build/bin

mv ./build/bin/r-type-server ./Server/r-type-server
mv ./build/bin/r-type-client ./Client/r-type-client

On Windows, don't forget to move all .dll with executable for client

mv bin/*.dll ../Client/r-type-client

Use the executables which has been created by make

For the server

./r-type_server

For the clients

./r-type_client
